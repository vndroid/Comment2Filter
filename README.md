# Comment2Filter

评论过滤插件，基于 [SmartSpam](https://www.yovisun.com/archive/typecho-plugin-smartspam.html) 开发，支持 Typecho 1.2 及以上版本。

## 插件亮点

感谢原作者 [@YoviSun](https://www.yovisun.com) ，在此基础上修复 Typecho 新版本的兼容性问题。

- 遵循 Typecho 1.2 开发规范，兼容性更好
- 支持自定义过滤规则，满足不同用户的需求
- 轻量级设计，不会对博客性能造成明显影响
- 支持过滤垃圾评论、敏感词评论等多种类型的评论
